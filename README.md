# MSRIT_IDEATHON
🚀 Similar Document Template Matching Algorithm

This project presents a multi-format document compliance analyzer that evaluates how closely a document follows a predefined structure and formatting.

The system focuses on template matching, not content similarity, and works across different document types.

🎯 Features
Supports DOCX, PDF, and PPTX formats
Extracts document structure (headings, sections, slides)
Performs formatting analysis (basic font, layout consistency)
Compares documents with template structure
Generates compliance score and detailed feedback
Flexible matching (e.g., Intro ≈ Introduction)
⚙️ How It Works
Upload template and target document
Parse document based on format
Extract structural and formatting features
Convert into a unified representation
Compare using matching algorithm
Generate score and highlight issues
🧠 Key Idea

The system introduces a unified structure abstraction layer, allowing documents from different formats to be analyzed in a common representation for accurate comparison.

🔥 Future Scope
Automatic document type detection
AI-based semantic validation
No need for manual template upload
Auto-correction of formatting issues
🎬 Demo

The current prototype demonstrates template-based comparison to validate structure and formatting compliance.
